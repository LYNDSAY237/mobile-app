Running the project

If you want video tutorials for opening the project, please send an email includes your item purchase code to prodevelopersteam@gmail.com with the subject “Video tutorial” to send you video tutorials. 

If you want to debug and run the project on the PC with live editing, 

1- Download and install Node.js

To run React Native projects on the computer, you need Node.js. Simply Download the latest version of the Node.js and install it.

2- Download and install JDK

3- Download Android SDK.

After the SDK downloaded, extract it into one of your local drives directly. Example: I:\SDK

4- Edit and add environment variables

Go to “System properties” from the start menu and navigate to the “Advanced” tab and click on “Environment Variables” and create System Variables with the following values:

ANDROID_HOME with the address of the SDK folder.

JAVA_HOME with the address of the JDK installation folder

Click on Path and click on the “edit” button. In the opened window, click on the “New” and create a path that is related to the “bin” folder located in the JDK folder.



 


 



 

5- Download Yarn and install it. (Don’t need to change parameters in the installation steps.)

6- Download and install Git (Don’t need to change parameters in the installation steps.)

7- Download and install VirtualBox

8- Download and install Genymotion

after that create a Genymotion account with a personal license.


 


 


 


 


 

 

Create a new device (Pixel 2 preferred) with Android 9 (API 28) and wait for downloading the information. After that start the device. If the device runs, it is ok. It will take longer to load for the first time. We suggest restarting your PC before running the emulator.

9- Install IDE like PhpStorm

Before anything, we suggest restarting your computer.

Create an empty React Native project.

 

Close the created project.

Open the project with IDE (Proacademy folder)

When the project loaded, check Node and Yarn.

Insert the following command in the command window: (Make sure that command start is in the project folder. We put commands in a text file for easy usage.)

yarn -v

If you see the yarn version, it is ok.

After that try another command:

node -v

If you see node version, it runs.

 

Now, insert the following command in the command line:

yarn install

Wait until all packages and dependencies downloaded.

Make sure Genymotin running and the device started.

Insert the below command in the command window to run the live preview of the project.

cd android && gradlew clean && cd ../ && npx react-natvie run-android

 

The project should be run on the emulator screen.

If you faced error, you can try this command:

del %appdata%\Temp\react-native-* & cd android & gradlew clean & cd .. & del node_modules/ & npm cache clean --force & yarn install & react-native run-android
